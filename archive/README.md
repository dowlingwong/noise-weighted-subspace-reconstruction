# Archive

Historical and superseded work lives here so the repository root stays focused
on runnable paper workflows.

- `equivalence/`: Paper 1 OF/EMPCA and linear-AE equivalence studies.
- `PC_interpretation/`, `PC_component_check/`: older component-interpretation
  notebooks and demo artifacts.
- `make_template/`: template and SNR-weight construction notebooks.
- `legacy_notebooks/experiments/`: pre-cleanup top-level experiment notebooks.
- `tutorials/`: standalone tutorials not tied to the active paper workflows.

Prefer moving old exploratory notebooks here rather than leaving new scaffold
folders at the repository root.
