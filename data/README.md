# Data Policy

Large public datasets do not live in this repository. The remote-server
default is `/ceph/dwong/paper1_dataset`, configurable through
`PAPER1_DATA_ROOT` or `--data-root`.

The committed files under `data/` are legacy small spectra, templates, and
exploratory assets retained for provenance. New GWOSC, CRESST, and TIDMAD data
must not be added here.
