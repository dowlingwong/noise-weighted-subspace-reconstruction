# TraceSimulator

`TraceSimulator` is a python module designed for generating and simulating DELight traces based on a given configuration. This module is intended to be used on the IAP-based `kalinka` computing system, where the necessary input files are located.

## Installation

To install the module, run the following command:

```sh
pip install .
```

This will install TraceSimulator and allows you to use it in your work. In case you are developing the module, you can install it using `pip install -e .`: this option installs the module in editable mode, allowing for the local changes to take immediate effect without the need to reinstall the package. The package can be uninstalled using `pip uninstall TraceSimulator`.

## Usage

An example of the usage is given in the `tutorials/Example.ipynb` notebook.
