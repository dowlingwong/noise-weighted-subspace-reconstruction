# Archive Policy

The archive holds reviewed stale Paper 1 scripts, notebooks, and generated
outputs only after useful logic has been migrated and provenance has been
recorded.

The first rebuild pass is intentionally conservative. `paper2/`, `NPML/`,
`TraceSimulator/`, and `QP_simulator/` remain in their existing locations.
Nothing should be moved here solely to make the top-level tree look cleaner.

- `stale_scripts/`: obsolete entry points after replacement.
- `old_docs/`: superseded plans retained for provenance.
- `old_notebooks/`: exploratory notebooks with no remaining execution role.
- `old_outputs/`: generated outputs retained for provenance.
