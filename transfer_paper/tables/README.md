# Paper-facing tables

This directory contains compact claim and result tables intended for direct
review by the paper-writing agent. Numerical tables generated from evidence
live under `transfer_paper/data/derived/`; this directory is reserved for
curated paper-facing selections and captions.

Current curated tables:

- `claim_status.csv` — allowed/prohibited statement control.
- `paper_implications.csv` — result-to-manuscript implication map.
- `method_traceability.csv` — experiment/protocol/output traceability.
- `figure_captions.md` — draft captions for available and pending figures.
